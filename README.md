# The Dog Park

## Purpose 
The purpose here was to collaboratively create a front end webiste that uses a music and movie api to generate the information of a movie and its related soundtrack with one search

## Built With
* HTML
* CSS
* JavaScript 
* Bootstrap
* MySQL
* Sequelize
* Node.js
* Express.js
